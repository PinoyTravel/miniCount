h1. miniCount jQuery Plugin (__written in CoffeeScript__)

The Letter/Word/Sentence Counter jQuery Plugin you've always been looking for.

miniCount is very basic but powerful Letter/Word/Sentence jQuery plugin and it just Works!

If you have any question of find a bug, please raise it <a href="https://github.com/MatthieuA/miniCount/issues">at GitHub</a> or <a href="http://www.twitter.com/matthieu_tweets">on Twitter</a>.
